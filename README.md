# proyecto_sprint_7
Proyecto sprint 7
Proyecto Sprint 7: Dashboard de Anuncios de Coches

Este proyecto es un ejercicio práctico de ingeniería de software que implica la creación y el despliegue de una aplicación web interactiva (dashboard) utilizando Streamlit.

Descripción de la Aplicación

El Dashboard de Análisis de Anuncios de Coches permite a los usuarios explorar visualmente un conjunto de datos de anuncios de venta de vehículos.

Funcionalidades

La aplicación proporciona las siguientes visualizaciones clave, generadas bajo demanda mediante casillas de verificación (checkboxes):

Histograma de Millaje (Odometer): Muestra la distribución de la variable de millaje en el conjunto de datos.

Gráfico de Dispersión (Precio vs. Millaje): Muestra la relación entre el precio de venta de un vehículo y su millaje, permitiendo al usuario identificar tendencias o posibles valores atípicos.

Tecnologías Utilizadas

Python: Lenguaje de programación principal.

Streamlit: Framework utilizado para crear la aplicación web interactiva de forma sencilla.

Pandas: Usado para la manipulación y limpieza inicial de datos.

Plotly: Librería utilizada para generar gráficos interactivos y visualmente atractivos.

GitHub/Render: Plataformas utilizadas para el control de versiones y el despliegue continuo.
